# Constructions Mod

Мод строительства баз для Minecraft 1.20.1 Forge. Интегрирует механику базостроения из игры RUST в Minecraft с поддержкой режима Adventure.

## Особенности

- 🏗️ **Система строительства RUST-стиля** - Размещение структур с превью голограмм
- 🔐 **Система авторизации** - Управление доступом через auth cabinet
- 💪 **Система здоровья** - Структуры имеют прочность и могут быть повреждены
- 💣 **Динамическая взрывчатка** - 3 типа взрывчатки разной мощности
- 🌍 **Адаптация под ландшафт** - Структуры подстраиваются под рельеф
- 🔗 **Скрепление структур** - Объединение связанных структур в единое целое
- 📡 **Сетевая синхронизация** - Полная синхронизация между клиентом и сервером

## Минимальные структуры

- ✅ Фундамент (5×5) с опорами
- ✅ Стена (5×4)
- ✅ Дверной проём (5×3)
- ✅ Потолок/Крыша
- ✅ Деревянная дверь (с авторизацией)
- ✅ Железная дверь (в 2 раза крепче)
- ✅ Сундук (5 дней жизни без auth cabinet)
- ✅ Шкаф авторизации (40 блоков радиус)
- ✅ Костёр (2 часа жизни без auth cabinet)

## Требования

- Minecraft 1.20.1
- Forge 47.4.10+
- Java 17+

## Установка

1. Скомпилировать мод:
```bash
gradle build
```

2. Найти JAR файл в `build/libs/constructions-*.jar`

3. Поместить в папку `mods` сервера/клиента

## Конфигурация

Основной конфиг находится в `config/constructions-server.toml`

```toml
# Минимальное расстояние от зон лута
minDistanceFromLoot = 100

# Максимальная высота столбов фундамента
maxFoundationHeight = 3

# Время жизни структур без auth cabinet (часов)
unclaimedStructureLifetime = 72

# Радиус действия auth cabinet
authCabinetRadius = 40

# Урон взрывчатки
weakExplosiveDamage = 2.0
mediumExplosiveDamage = 5.0
strongExplosiveDamage = 10.0
```

## Использование

### Размещение структуры

1. Возьмите в руки предмет структуры (например, фундамент)
2. Посмотрите на землю - увидите голограмму превью (синяя = можно, красная = нельзя)
3. Нажмите ПКМ для размещения

### Удаление структуры

1. Возьмите молоток строителя
2. Нажмите ЛКМ + Shift на свою структуру
3. Структура будет удалена

### Урон структурам

1. Возьмите взрывчатку и прикрепите её к структуре (ПКМ)
2. Активируйте взрывчатку
3. Структура получит урон
