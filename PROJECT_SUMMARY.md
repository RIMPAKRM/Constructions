## 🎮 Игровая логика

### Размещение структуры:
1. Игрок в режиме Adventure берет предмет
2. Видит голограмму превью (синяя = можно)
3. Проверяются условия (расстояние от лута, авторизация)
4. Структура размещается и синхронизируется

### Урон структурам:
1. Взрывчатка прикрепляется к структуре
2. Активируется - создает взрыв
3. Структура получает урон
4. При 0 здоровья - структура разрушается

### Авторизация:
1. Auth cabinet создает защищённую зону
2. Игроки могут быть авторизованы
3. Двери открываются только авторизованными
4. Сундуки открываются всеми
