## Ошибки и исправления

### Ошибка в ModNetworking.java

**Строка 71-73:**
```java
public static void sendToAll(Object message) {
    INSTANCE.send(com.net.minecraft.server.players.PlayerList.);
}
```

**Исправление:**
```java
public static void sendToAll(Object message, ServerPlayer... players) {
    for (ServerPlayer player : players) {
        sendToPlayer(message, player);
    }
}
```

Или использовать `PacketDistributor` из ForgeNetworking.
